# 振興三倍券郵局地圖 🗺 - Mac 版本

## 說明

- 振興三倍券是台灣政府於 2020 年為了因應肺炎疫情造成的經濟緊縮，所發放的消費專用券。
- 本 App 的功能是顯示台灣郵局的三倍券即時領券量，資料集來源為 [https://data.gov.tw/dataset/127751](https://data.gov.tw/dataset/127751)
- 可用期限：2020/07/15 ~ 2020/12/31
